🎓 Student Management System (Python)

A console-based Student Management System built using Python and OOP concepts.
It allows users to manage student records with persistent file storage.

🚀 Features

- Add student
- View all students
- Search student by roll number
- Delete student
- Data stored in file (students.txt)

🛠️ Tech Stack

- Python
- File Handling
- OOP Concepts

▶️ Run

python main.py

📂 Project Structure

main.py
students.txt
README.md

💡 Key Highlights

- Uses OOP for structured design
- Implements CRUD operations
- Persistent storage using file handling
